vbox-sharefolder

Posted on 2018-09-01

vbox虚拟机centos系统，与宿主机windows系统共享文件夹

centos配置

1.需要用到VBoxGuestAdditions,到地址找到适合的版本（最新版）

2.下载iso镜像文件，然后将镜像挂载到centos的存储驱动中（设置-存储-控制器ide）

3.在centos中将镜像挂载到本地，mount /dev/cdrom /mnt/cdrom

4.进入/mnt/cdrom/，执行 ./VBoxLinuxAdditions.run

5.将windows的共享目录挂载到vbox（设置-共享文件夹-固定分配）,设置名称xxx

6.在centos中将windows目录挂载带本地,执行 mount -t vboxsf xxx /mnt/xxx

遇到问题

1./VBoxLinuxAdditions.run 安装失败，提示内核不匹配

更新内核yum update kernel，yum install gcc ，yum install kernel-devel ，重启

2.安装一直卡住

换其他VBoxLinuxAdditions

3.执行 mount -t vboxsf xxx /mnt/xxx 提示 no such device

VBoxLinuxAdditions没有安装成功
