php5.4升级到php5.6

http://blog.csdn.net/ebw123/article/details/41875255

先把 php-fpm服务关闭（其实不影响的，因为程序在内存里面跑 不关也没事的） 然后 先把 /usr/local/php 重命名 以防止如果新版本更新失败 回滚

mv /usr/local/php /usr/local/php5.3.8

wget http://www.php.net/distributions/php-5.6.3.tar.gz

tar -zxvf php-5.6.3.tar.gz

编辑Makefile 大约77 行左右的地方: EXTRA_LIBS = ….. -lcrypt 在最后加上 -liconv

make

make install

编辑 php-fpm.conf

cp sapi/fpm/init.d.php-fpm /etc/init.d/php-fpm

chmod +x /etc/init.d/php-fpm

chkconfig –add php-fpm

chkconfig php-fpm on

重启php-fpm
