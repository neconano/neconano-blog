目的：降低学习难度（业务熟悉快，代码易懂，新人也能快速上手），提高代码性能和稳定性（简单不易出错）

1.docker demo

现在可以通过docker来拉取一个最新的SD环境和服务器。
拉取SD的运行环境：docker pull registry.cn-hangzhou.aliyuncs.com/tmtbe/sd-swoole
直接拉取一个最新的SD服务器：
docker pull registry.cn-hangzhou.aliyuncs.com/tmtbe/sd-server
拉取后可以直接运行：
docker run -P -d –name sd registry.cn-hangzhou.aliyuncs.com/tmtbe/sd-server
访问即可看到hello页面

通过docker port sd看端口映射
通过数据卷映射本机的代码到docker中：
docker run -d -P –name sd -v /home/SwooleDistributed/src:/home/SwooleDistributed/src registry.cn-hangzhou.aliyuncs.com/tmtbe/sd-server
在容器中无法访问挂载卷问题，需要执行
chcon -Rt svirt_sandbox_file_t /home/SwooleDistributed/src

修改了本机的代码，docker中会立即reload
