build

Swoole扩展是按照php标准扩展构建的。使用phpize来生成php编译配置，./configure来做编译配置检测，make进行编译，make install进行安装。

config.m4

利用config.m4文件，执行phpize即可生成configure和make文件，执行./configure -help可以查询在config.m4中定义的选项

http://www.php.net/manual/zh/internals2.buildsys.configunix.php

config.m4文件用于指定正在开发的扩展在类unix系统下构建时支持的选项，指定此扩展需要哪些库以及哪些源文件；使用 GNU autoconf 语法编写。
注意需要重新执行phpize，config.m4的修改才会生效；
在执行./configure时，所有的输出将记录到config.log里，通过查看此文件可以调试config.m4。
