安装学习环境（windows）

1.安装vagrant和virtualbox

安装后再cmd执行vagrant显示版本表示正常

2.下载box镜像

3.加载镜像

在镜像所在目录cmd执行

vagrant box add ubuntu16 virtualbox.box

生成Vagrantfile

将样例中的配置文件复制替换样例

修改项目box=”virtualbox.box”

生成ssh密钥

执行ssh-keygen,根据配置文件输入ubuntu作为登陆密钥(ssh-keygen需要安装git，在环境变量中引入usr/bin路径)

依次执行vagrant up,vagrant provision,vagrant ssh

启动virtualbox,并登录成功

4.安装docker（vagrant区别的另一种安装方案）

参考

5.使用docker加载镜像

需要注意使用root账号

sudo su

加载镜像（该镜像可能在docker hub的公共库中）

docker pull dongweiming/web_develop:dev

进入容器

docker run –name web_dev -it -p 9000:9000 -p 3141:3141 -p 5000:5000 dongweiming/web_develop:dev /bin/zsh

执行exit可以退出

启动容器

docker start web_dev

重新进入容器

docker attach web_dev

6.pip安装

sudo apt-get install python-pip -yq

sudo pip install pip -U -q
