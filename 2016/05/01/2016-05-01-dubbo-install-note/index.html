dubbo安装笔记

根据dubbo在git上的文档安装‘文档’

1.安装maven

安装maven的目的应该是生成eclipse的项目用

文档中的下载地址已经失效，在‘版本目录’中选择一下载

ubuntu系统添加环境变量，

vi /etc/profile

追加

export PATH=/newdata/www-src/apache-maven-3.2.2/bin:$PATH

执行

source /etc/profile

2.导入dubbo到eclipse项目（适用windows环境）

cd ~/dubbo

mvn eclipse:eclipse

提示错误

提示连接不上,去git上下载opensesame，进入目录执行 mvn install 成功后问题解决

根据网上建议使用mvn clean install -Dmaven.test.skip方式编译带样例版本

继续报错

在maven配置文件settings.xml中添加镜像(如果不好用百度其他的)

其他方案中，可以下载编译依赖直接解压放到~/.m2/repository/com(/root/.m2/repository/com)目录中再编译

再执行后提示错误

修改对应pom.xml的依赖

再执行后错误

应该是内存溢出，调整几个参数后也没有效果（在windows下没有问题）

将执行命令调整为 mvn install -Dmaven.test.skip 执行后成功

其他问题处理方案

3.安装zookeeper

http://mirrors.cnnic.cn/apache/zookeeper/zookeeper-3.4.8/zookeeper-3.4.8.tar.gz

参考

cp conf/zoo_sample.cfg conf/zoo.cfg

vi conf/zoo.cfg

dataDir=/home/dubbo/zookeeper-3.3.3/data

./bin/zkServer.sh start
