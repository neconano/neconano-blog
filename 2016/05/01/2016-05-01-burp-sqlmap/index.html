burp-sqlmap笔记

1.burp 安装

软件收费，找了个能用的地址

安装jdk后，在目录页面执行 java -jar BurpLoader.jar

2.使用burp抓包

1.配置浏览器代理

打开火狐浏览器选项-高级-网络-连接设置参考

2.burp抓包使用

打开浏览器，在输入想要抓取的页面，开始的时候页面会卡主，等待burp的返回，

选择proxy标签，在intercept项目中，点击forward返回浏览器进行下一步，过程中会显示抓取的内容

3.sqlmap安装

下载sqlmap,最新版本即可地址

安装jdk等以来后，安装python2版本，将java,sqlmap,python加入windows系统环境变量即可

4.sqlmap使用

将抓包获得的数据保存下来，命名如123

在命令行执行 sqlmap.py -r 123

1.1.使用sqm.pyw

下载，解压后覆盖到sqlmap目录中，运行sqm_cn.pyw

如有问题，安装ttk,使用cmd在pyttk-0.3-py3k执行setup.py install 命令。

使用方法

目标选择执行的url或文件，选择参数后点击get query,点击start执行

相关注入参数sqlmap教程

  1. –flush-session 刷新缓存

  2. –dbs 列出数据库
