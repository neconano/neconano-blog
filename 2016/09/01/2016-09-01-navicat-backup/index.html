使用navicat备份数据库

1.备份库

使用navicat的备份功能即可

2.备份视图

1.选择以生成的备份.psc

2.点击–提取SQL–保留视图–导出sql

3.还原备份

1.在还原中提示1067. Invalid default value for xxx

则检查xxx表的下一个表，是否有字段非法，如类型默认值，例如dateline不允许设置默认值

2.还原表的时候，不要勾选视图，视图有依赖关系会导致失败

3.还原视图

将导出的试听sql导入即可

使用mysqldump导出

1.导出整个数据库

mysqldump -u 用户名 -p 密码 -h 远程IP –databases 数据库名 > 导出的文件地址

mysqldump -u dbuser -p -h 127.0.0.1 –databases dbname > c:/dbname.sql

使用mysql导入

进入mysql数据库控制台，如

mysql -u root -p

mysql>use 数据库

然后使用source命令，后面参数为脚本文件(如这里用到的.sql)

mysql>source d:/dbname.sql

使用xtrabackup备份

1.下载安装

下载地址

官方推荐yum方式

yum list | grep percona

yum install percona-xtrabackup-22

2.备份全库参考文章

innobackupex –user=DBUSER –password=DBUSERPASS /path/to/BACKUP-DIR/

innobackupex –apply-log /path/to/BACKUP-DIR

3.备份部分库

innobackupex –databases=”mageedu testdb” /path/to/backup
