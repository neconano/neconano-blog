hexo github 笔记

Posted on 2016-09-01

使用hexo管理github page

1.安装hexo

参考

1.需要安装git

2.需要安装Node.js

安装完成后，在项目目录执行

npm install -g hexo-cli

生成hexo

然后依次执行下面的命令

hexo init

npm install

安装成功后，启动hexo预览生成

hexo server

2.安装hexo主题

下载next

将主题放入到 hexo/theme 目录中

官方文档

3.将hexo站点同步到github

修改hexo的配置文件_config.yml

deploy:
type: git
repository: https://username:password@github.com/neconano/neconano.github.io.git
branch: master

在hexo根目录执行

同步完成

如果提示 error deployer not found 执行如下

npm install hexo-deployer-git –save

4.图片上传

1.hexo根目录执行

npm install https://github.com/CodeFalling/hexo-asset-image –save

2.配置_config.yml

post_asset_folder: true

在_post目录下创建文章同名目录，将图片放入即可
