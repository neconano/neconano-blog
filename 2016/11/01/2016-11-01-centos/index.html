centos 7

本地测试可使用vbox,网络上选择桥接

1.设置ip地址

vi /etc/sysconfig/network-scripts/ifcfg-enp0s3

确保192.168.1.240有效

重启服务ifup ifcfg-enp0s3

查看ip地址

ip addr

2.防火墙

sudo systemctl stop firewalld.service && sudo systemctl disable firewalld.service

3.调整yum源

nginx

centos7系统库中默认是没有nginx的rpm包

rpm -Uvh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm

yum install nginx

service nginx start

使用killall

yum install psmisc

php

这个版本是5.4

安装5.6

以下是CentOS 7.0的源。

ImageMagick and imagick

图片处理功能，可将pdf等格式转成其他图片格式

ImageMagick是个软件，在安装的时候，查看configure是否支持png等你需要的格式

安装完成后，可以使用如下命令将pdf转成jpg，测试是否成功

convert -density 300 -trim test.pdf -quality 100 test.jpg

imagick是php扩展

最好使用pecl方式安装，否则可能会提示处理的图片不支持

JDK

不推荐使用Yum安装的jdk（没有javac命令）

查询端口和进程

查询端口占用

lsof -i:端口

查询进程占用端口

netstat -apn|grep 进程名

查询进程id

ps aux|grep 进程名
