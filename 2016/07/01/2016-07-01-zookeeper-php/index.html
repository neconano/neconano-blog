zookeeper 笔记

1.centos安装jdk

1.查看CentOS自带JDK是否已安装。

yum list installed |grep java

若有自带安装的JDK,卸载，例如 yum -y remove java-1.7.0-openjdk*

2.查看yum库中的Java安装包。

yum -y list java*

3.使用yum安装Java环境

yum -y install java-1.8.0-openjdk*

4.查看刚安装的Java版本信息。

java -version
