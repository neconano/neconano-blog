  • 创建Deploy Keys

1.打开git shell

2.使用命令ssh-keygen –C “邮箱地址” –t rsa生成key

3.打开生成的key,复制内容，打开github.com上创建的repository，点旁边的edit，在Deploy Keys里点add another deploy key，名字随便起，把刚才复制的内容粘贴在上面。

4.在项目的setting里面，可以看到页面是否被正确编译，在文件的history中也可以看到
